# Сортировка массива методом пузырька

## Описание проекта

Данный проект представляет собой консольное приложение на языке C, реализующее сортировку массива **методом пузырька** (
Bubble Sort). Приложение позволяет:

- Сортировать массив из текстового файла
- Генерировать случайный массив заданного размера
- Сохранять отсортированный массив в файл
- Выбирать направление сортировки (по возрастанию/убыванию)
- Запускать автоматические тесты для проверки корректности работы

---

## Алгоритм сортировки пузырьком

### Принцип работы

**Сортировка пузырьком** — это простой алгоритм сортировки, который многократно проходит по массиву, сравнивая соседние
элементы и меняя их местами, если они стоят в неправильном порядке. Проходы по массиву повторяются до тех пор, пока
массив не будет полностью отсортирован.

### Сложность алгоритма

| Характеристика                   | Оценка |
|----------------------------------|--------|
| Лучший случай (уже отсортирован) | O(n)   |
| Средний случай                   | O(n²)  |
| Худший случай (обратный порядок) | O(n²)  |
| Память                           | O(1)   |

## Реализация алгоритма в коде

```c
void bubbleSort(int *arr, int size, int reverse)
{
   clock_t start = clock();
   int tmp;
   int numberOfPermutations;
   int numberOfExchanges;

   numberOfPermutations = 0;
   do
   {
      numberOfExchanges = 0;
      for (int i = 0; i < size - 1; i++)
      {
         if (!reverse && arr[i] > arr[i + 1])
         {
            tmp = arr[i];
            arr[i] = arr[i + 1];
            arr[i + 1] = tmp;
            numberOfExchanges++;
         } else if (reverse && arr[i] < arr[i + 1]) {
            tmp = arr[i];
            arr[i] = arr[i + 1];
            arr[i + 1] = tmp;
            numberOfExchanges++;
         }
      }
      size--;
      numberOfPermutations += numberOfExchanges;
   } while (numberOfExchanges > 0); // Если обменов не было, то массив отсортирован

   clock_t end = clock();
   double time_spent = (double)(end - start) / CLOCKS_PER_SEC;
   printf("\nВремя сортировки: %.6f секунд\n", time_spent);
   printf("Количество перестановок: %d\n", numberOfPermutations);
}
```

## Объяснение ключевых моментов

### Параметр reverse — определяет направление сортировки:

- reverse = 0 — по возрастанию (от меньшего к большему)

- reverse = 1 — по убыванию (от большего к меньшему)

Оптимизация size-- после каждого прохода самый большой (при сортировке по возрастанию) или самый маленький (при
сортировке по убыванию) элемент гарантированно оказывается в конце, поэтому диапазон сравнения уменьшается.

Условие numberOfExchanges > 0 — если за проход не было произведено ни одного обмена, значит массив уже отсортирован и
алгоритм завершается досрочно.

Подсчёт времени — замеряется фактическое время выполнения сортировки с использованием clock().
